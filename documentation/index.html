PasteShr - Text hosting and sharing script

Pasteshr is a script which allows you to store any text online for easy sharing. The idea behind the script is to make it more convenient for people to share large amounts of text online.

Requirements

Make sure you have:
  • A web server which you can access via FTP or SFTP.
  • A text editor.
  • A web browser.
  • OpenSSL PHP Extension
  • PDO PHP Extension
  • Mbstring PHP Extension
  • Tokenizer PHP Extension
  • XML PHP Extension
  • Ctype PHP Extension
  • JSON PHP Extension
  • BCMath PHP Extension
  • cURL PHP Extension
And make sure your web server is running:
  • Web serving software such as Apache. PHP 7.1.3 or later.
  • MySQL 5.x for best performance.
If you are not sure about this, please check with your web hosting provider.

Installation

Easy (Web) Installation -

1 - Create a MySQL database, and a MySQL User with full permissions for that database.

2 - Note down the MySQL details: username, password, database name and server host name. If MySQL is running on the same server as your website, the server host name is likely to be 127.0.0.1 or localhost.

3 - Place all the files from "upload_files" folder [Recommended] Or if you want to install PasteShr inside a sub directory then all the files from "upload_files_subdir_version" (including .htaccess) to the appropriate location (public_html or www) on your web server

Set file permissions as follow

/bootstrap        775
/public/uploads   775
/storage          775

4 - Set your domain or subdomain's DOCUMENT ROOT to /<path to public_html>/public

For dedicated server you can configure Apache virtual host as follows

<VirtualHost *:80>
	ServerName yoursite.com
	DocumentRoot "/<path to pasteshr>/public"
	Options Indexes FollowSymLinks
	<Directory "/<path to pasteshr>/public">
		AllowOverride All
		Require all granted
	</Directory>
</VirtualHost>

5 - Open the appropriate web page in your web browser, for example

  • http://mysite.com/install

6 - Follow the web installer instructions.

if installer doesn't work follow the manual installation.

Manual Installation -

1 - Create a MySQL database, and a MySQL User with full permissions for that database.

2 - Note down the MySQL details: username, password, database name and server host name. If MySQL is running on the same server as your website, the server host name is likely to be 127.0.0.1 or localhost.

3 - Place all the files from "upload_files" folder [Recommended] Or if you want to install PasteShr inside a sub directory then all the files from "upload_files_subdir_version" (including .htaccess) to the appropriate location (public_html or www) on your web server

Set file permissions as follow

/bootstrap        775
/public/uploads   775
/storage          775

4 - Open .env in your text editor, set the following details, and save the file. Do not use a word processor such as Microsoft Word for this, but rather Notepad or another appropriate text editor software.

APP_URL=MY_URL
DB_HOST=MY_DBHOST
DB_DATABASE=MY_DBNAME
DB_USERNAME=MY_DBUSERNAME
DB_PASSWORD=MY_DBPASSWORD

5 - Set your domain or subdomain's DOCUMENT ROOT to /<path to public_html>/public

For dedicated server you can configure Apache virtual host as follows

<VirtualHost *:80>
	ServerName yoursite.com
	DocumentRoot "/<path to pasteshr>/public"
	Options Indexes FollowSymLinks
	<Directory "/<path to pasteshr>/public">
		AllowOverride All
		Require all granted
	</Directory>
</VirtualHost>

6 - Open the appropriate web page in your web browser, for example:

  • If you installed the script at root folder of a domain, http://www.mysite.com/

7 - To set up your administrator account, navigate  http://www.mysite.com/admin/login

FAQ

  • Where can I access the admin panel?

    Open yoursite.com/admin/login in your browser and login to using your admin credentials.

  • Where can I modify the HTML tamplates?

    You can find the layout files in ./resources/views/front/ The files extention is .blade.php

  • Where can I edit the stylesheet?

    You can edit the stylesheet from ./public/css The main style file is: style.css

  • Where can I add my own code inside head/body/footer tags?

    You can do that from /resources/views/front/includes/ & /resources/views/front/layouts/default.blade.php

  • Where is favicon.ico located?

    You can change it from ./public/favicon.ico

  • How can I edit the footer menu?

    You can edit the footer from: ./resources/views/front/includes/footer.blade.php

  • How can I setup Google invisible reCaptcha?

    You have to get a site key from https://www.google.com/recaptcha/ Once you got the SiteKey & SecretKey, you can add it in Admin Panel > Settings > Invisible Recaptcha Settings.

  • How can I setup Advertisement blocks?

    You can add it in Admin Panel > Settings > Advertisement Settings.

  • How can I add a new site language?

    You can do that from Admin Panel > Site Languages > Create

  • How can I translate a new site language?

    You can do that from Admin Panel > Site Languages > Translations

  • How can I set a default site language?

    You can do that from Admin Panel > Settings > Default Site Locale

  • How can I set default site timezone?

    You can do that from Admin Panel > Settings > Default Timezone
